## Clan Manager webapp (ES6 Experiment)

### Initial setup

```bash
# Clone the repo...
git clone https://github.com/darkowlzz/senso.git
cd senso

# Then, you need to install all the dependencies...
npm i

# Install gulp
npm install -g gulp
```

### Running in the browser
```bash
gulp build
gulp serve

# If you want Gulp to re-build on every change...
gulp watch
```
